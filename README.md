# Honga Protfolio Website

## Update Repository

```powershell
./update-profolio.ps1
```

## Update Page

Install dependencies

```bash
pnpm install
```

Run Build

```bash
pnpm build
```

Deploy

```bash
pnpm run deploy
```

## Develop

Run Develop

```bash
# local debug
pnpm dev

# local/mobile debug
pnpm dev:host
```
